# PDI — Plano de Desenvolvimento Individual
**Disciplina:** Algoritmos Avançados  

---

## 1. Objetivo do Semestre
Minha principal motivação é evoluir tecnicamente para aplicar algoritmos mais eficientes no meu dia a dia como Analista de Projetos na NeoCredit. Quero dominar a análise de complexidade e estruturas de dados avançadas para garantir que as integrações e análises que gerencio para clientes de grande porte sejam escaláveis. Meu foco é migrar de uma programação funcional básica para uma visão de engenharia de software mais otimizada.

## 3. Metas Técnicas

### Meta 1: Análise de Performance em Projetos Reais
* **Contexto:** Importante para avaliar o impacto de novas rotinas de análise de crédito.
* **Descrição:** Realizar a análise de complexidade de tempo de todos os algoritmos autorais desenvolvidos.
* **Plano de ação:** Estudar notação Big O e documentar a complexidade em cada script.
* **Medição:** Comprovação através dos comentários técnicos nos códigos entregues.
* **Prazo:** Abril/2026.

### Meta 2: Aplicação de Programação Dinâmica
* **Contexto:** Reduzir o custo computacional em modelos de análise financeira repetitivos.
* **Descrição:** Resolver problemas de otimização utilizando técnicas de memoização.
* **Plano de ação:** Praticar 2 exercícios semanais de DP em plataformas de desafio.
* **Medição:** Repositório atualizado com soluções validadas.
* **Prazo:** Maio/2026.

### Meta 3: Domínio em Algoritmos de Grafos
* **Contexto:** Mapear fluxos de trabalho e dependências de processos de clientes.
* **Descrição:** Implementar um algoritmo de caminho mínimo (Dijkstra) funcional.
* **Plano de ação:** Estudo dirigido de grafos e implementação de um caso de uso real.
* **Medição:** Protótipo funcional rodando sem erros de lógica.
* **Prazo:** Junho/2026.

## 5. Plano de Uso de IA
Pretendo usar a IA como um **tutor pedagógico**.
* **Como:** Para explicar conceitos teóricos complexos e sugerir casos de teste para validar meus códigos.
* **Limites:** Não utilizarei a IA para gerar soluções completas. O objetivo é que ela ajude no debug e na compreensão da lógica, mas a autoria final do código será sempre minha.
